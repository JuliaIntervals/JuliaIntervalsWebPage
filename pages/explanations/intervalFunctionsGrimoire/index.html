    INTRODUCTION TO INTERVAL ARITHMETIC

    The purpose of this section is to give an overview of the theoretical background behind interval arithmetic. The material of this section is based on Introduction to Inverval Analysis from Moore, Kearfott and Cloud.

    1. Basic Concepts
    2. Basic Operations

    Basic Concepts

    The fundamental concept in interval arithmetic is the concept of interval. Given a lower bound X‾\underline{X}X​ and an upper bound X‾\overline{X}X, we define the interval XXX as the subset X⊆RX\subseteq\mathbb RX⊆R

    This is shortly denoted as X=[X‾, X‾]X = [\underline{X},~\overline{X}]X=[X​, X].

    Particularly, we require that X‾≤X‾\underline{X}\leq\overline{X}X​≤X, i.e. for example [3, 5][3,~5][3, 5], [−1, 3][-1,~3][−1, 3] are valid intervals, but [3,−1][3,-1][3,−1] or [5,3][5,3][5,3] are not. If X‾=X‾=x\underline{X}=\overline{X}=xX​=X=x, then the interval X=[X‾,X‾]=xX=[\underline{X},\overline{X}]=xX=[X​,X]=x corresponds to the real number xxx, in this case we say that the interval is degenerate.

    Basic Operations

    Given two intervals XXX and YYY and a binary operation ∘\circ∘, the results is generally defined as

    X∘Y={x∘y ∣ x∈X and y∈Y}. X\circ Y=\{x\circ y~|~x\in X \text{ and }y\in Y\}.X∘Y={x∘y ∣ x∈X and y∈Y}.

    Using this definition, the 4 basic arithmetic operations can be defined as follows

    X+Y=[X‾+Y‾, X‾+Y‾]X+Y=[\underline{X}+\underline{Y},~\overline{X}+\overline{Y}]X+Y=[X​+Y​, X+Y]
    X−Y=[X‾−Y‾, X‾−Y‾]X-Y=[\underline{X}-\overline{Y},~\overline{X}-\underline{Y}]X−Y=[X​−Y, X−Y​]
    X⋅Y=[min⁡S, max⁡S]X\cdot Y=[\min S,~\max S]X⋅Y=[minS, maxS], where S={X‾Y‾, X‾Y‾, X‾Y‾, X‾Y‾}S=\{\underline{X}\underline{Y},~\underline{X}\overline{Y},~\overline{X}\underline{Y},~\overline{X}\overline{Y}\}S={X​Y​, X​Y, XY​, XY}
    1X=[1X‾, 1X‾]\frac{1}{X}=\left[\frac{1}{\overline{X}},~\frac{1}{\underline{X}}\right]X1​=[X1​, X​1​] with 0∉X0\notin X0∈/​X
    XY=X⋅1Y\frac{X}{Y}=X\cdot\frac{1}{Y}YX​=X⋅Y1​ with 0∉Y0\notin Y0∈/​Y

    The definitions of our arithmetic operations can be easily derived from (1). Let us now investigate what properties these operations have.

    Similar to "normal" arithmetic operations, addition and multiplication are associative and commutative, and have 000 and 111 as identity element, respectively.

    Opposed to "normal" arithmetic, X−X≠0X-X\neq 0X−X​=0, indeed

    X−X=[X‾−X‾, X‾−X‾]X-X=[\underline{X}-\overline{X},~\overline{X}-\underline{X}]X−X=[X​−X, X−X​],

    now since X‾≤X‾\underline{X}\leq\overline{X}X​≤X, X−X=0⇔X‾=X‾X-X=0\Leftrightarrow\underline{X}=\overline{X}X−X=0⇔X​=X, i.e. if and only if XXX is a degenerate interval. However, it always hold that 0∈X−X0\in X-X0∈X−X.

    Similarly, we can verify that 1∈XX1\in\frac{X}{X}1∈XX​ always, but 1=XX1=\frac{X}{X}1=XX​ only if XXX is degenerate.

    Another important difference to traditional arithmetic, the distributive property does not hold. Instead, interval arithmetic is subdistributive, i.e.

    A⋅(B+C)⊆A⋅B+A⋅CA\cdot(B+C)\subseteq A\cdot B+A\cdot CA⋅(B+C)⊆A⋅B+A⋅C

    Let's visualize this with an example

    [1,3]⋅([1,2]+[−3,−2])=[1,3]⋅[−2,0]=[−6,0][1,3]\cdot([1,2]+[-3,-2])=[1,3]\cdot[-2,0]=[-6,0][1,3]⋅([1,2]+[−3,−2])=[1,3]⋅[−2,0]=[−6,0]

    and

    [1,3]⋅[1,2]+[1,3]⋅[−3,−2]=[1,6]+[−9,−2]=[−8,4][1,3]\cdot[1,2]+[1,3]\cdot[-3,-2]=[1,6]+[-9,-2]=[-8,4][1,3]⋅[1,2]+[1,3]⋅[−3,−2]=[1,6]+[−9,−2]=[−8,4]

    and we notice indeed that [−6,0]⊆[−8,4][-6,0]\subseteq[-8,4][−6,0]⊆[−8,4]. Let's take another example

    [1,2]⋅([1,3]+[2,4])=[1,2]⋅[3,7]=[3,14][1,2]\cdot([1,3]+[2,4])=[1,2]\cdot[3,7]=[3,14][1,2]⋅([1,3]+[2,4])=[1,2]⋅[3,7]=[3,14]

    and

    [1,2]⋅[1,3]+[1,2]⋅[2,4]=[1,6]+[2,8]=[3,14][1,2]\cdot[1,3]+[1,2]\cdot[2,4]=[1,6]+[2,8]=[3,14][1,2]⋅[1,3]+[1,2]⋅[2,4]=[1,6]+[2,8]=[3,14]

    in this no over estimate happens. Moreover, the distributive property A(B+C)=AB+ACA(B+C)=AB+ACA(B+C)=AB+AC holds if BBB and CCC have the same sign.
